# Study 01 — Timeline Custom Track

## 목적
Unity Timeline을 확장해 자체 Track/Clip/Behaviour를 구현하면서, 
PlayableAsset과 PlayableBehaviour의 구조·라이프사이클을 체득한다.

## 구현 계획
- **DialogueTrack**: 대사 출력 + 타이핑 효과
- **CameraShakeTrack**: Cinemachine Impulse 기반 연출
- **CharacterMoveTrack**: 시간축 기반 캐릭터 이동
